Test Commands

Once loaded, test these commands in SillyTavern chat:

Status Commands

/stres status - Shows extension status and backend connection

/stres help - Shows available commands

/stres campaign - Shows campaign information

Inventory Commands

/inventory show - Display current inventory

/inventory add sword 1 - Add item to inventory

/inventory remove sword 1 - Remove item from inventory

/inventory use potion - Use an item

Combat Commands

/combat status - Show combat status

/combat act attack goblin - Submit combat action

Troubleshooting

If Commands Don't Work

  1. Check browser console for error messages
  2. Look for [STRES] messages
  3. Verify extension is properly installed in SillyTavern
  4. Try refreshing the page

If Extension Doesn't Load

  1. Check that manifest.json has correct format
  2. Verify extension files are in the right location
  3. Check SillyTavern's extension directory structure
  4. Look for JavaScript errors in console
